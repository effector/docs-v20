# `VueEffector(Vue, options?)`

`effector-vue` plugin

#### Arguments

1. `Vue` (_class Vue_): Vue class
2. `options` (_Object_): Plugin options

#### Returns

(_`void`_)

#### Example

```js
import Vue from 'vue'
import {VueEffector} from 'effector-vue'

Vue.use(VueEffector)
```
